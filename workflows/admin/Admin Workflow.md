# Admin Layout Workflows

This document describes the workflows for each page in the Admin Layout of the HCM Digitalized Street Food system.

---

## 1. Dashboard Workflow

This workflow describes how an admin monitors system performance and financial metrics through the dashboard: viewing statistics, filtering data by date, and using quick filters.

**A. View System Statistics**

1. Navigate to the **Dashboard** page from the sidebar.
2. View summary cards for **Total Revenue**, **Total Compensation Amount**, **Registered Users**, and **Vendor Conversions**.
3. Observe graphical charts for daily sign-ups, revenue, compensation, and conversions.
4. Hover over the **(i) Information** icon or specific chart elements to see detailed tooltips and data point values.

**B. Filter Statistics by Date**

1. On the **Dashboard** page, locate the date filter section.
2. Select a start date in the **From date** input.
3. Select an end date in the **To date** input.
4. Click the **Apply** button to refresh the statistics and charts for the selected period.

**C. Quick Filter Statistics**

1. On the **Dashboard** page, locate the quick filter buttons.
2. Click **Last 7 days**, **Last 30 days**, or **Last 3 months** to instantly filter data for those periods.

**D. Access Guide Tour**

1. On the **Dashboard** page, click the **Question Mark (Help)** icon next to the page title.
2. Follow the interactive steps to learn about the dashboard features.

## 2. Verification Workflow

This workflow describes how an admin verifies and processes branch registration requests from vendors and reviewers: viewing requests, filtering by type, reviewing detailed documentation, and approving or rejecting applications.

**A. View Pending Registrations**

1. Navigate to the **Verification** page from the sidebar.
2. View the list of pending requests in the table, including branch name, owner, and registration date.

**B. Filter Registrations by Type**

1. On the **Verification** page, click on the tabs (**Reviewer shared**, **Branch registration**, or **Ownership request**) to filter the requests by their source or type.

**C. View Registration Details**

1. Click the **Eye (Visibility)** icon in the actions column of a request.
2. Review the detailed information in the modal, including contact information, branch details, and descriptions.

**D. View License / Images**

1. Click the **License** icon to view the uploaded business license document.
2. Click the **Image** icon to view photos of the branch.

**E. Approve Registration**

1. Click the **Check Circle** icon in the actions column of a request.
2. In the confirmation dialog, click **Confirm** to verify the registration.

**F. Reject Registration**

1. Click the **Cancel** icon in the actions column of a request.
2. In the **Reject Reason** modal, enter the reason for rejection in the text field.
3. Click the **Confirm** button to complete the rejection.

**G. Access Guide Tour**

1. On the **Verification** page, click the **Question Mark (Help)** icon next to the page title.
2. Follow the interactive steps to learn about the verification process.

## 3. Vendor Management Workflow

This workflow describes how an admin manages the list of registered vendors in the system: viewing vendor details, checking branch statuses, reviewing business licenses, and suspending or reactivating accounts.

**A. View Vendor List**

1. Navigate to the **Vendors** page from the sidebar.
2. Browse the list of vendors, including their names, owners, and active statuses.

**B. View Vendor Detail**

1. Click the **Eye (Visibility)** icon in the actions column of a vendor.
2. Review the **General Information** card for vendor name, owner name, and creation date.
3. Scroll to the **Branches** section to see all branches owned by this vendor.

**C. View Branch Feedback**

1. In the **Vendor Detail** modal, locate a specific branch in the list.
2. Click the **View Feedback** button to open a modal showing user reviews and ratings for that branch.

**D. View Branch Location / License**

1. In the **Vendor Detail** modal, scroll to a branch's information section.
2. View the **Vị trí trên bản đồ** map preview to see the branch location.
3. View the **Giấy phép kinh doanh** section to see license images and status.

**E. Suspend Vendor**

1. Click the **Block** icon in the actions column for an active vendor.
2. In the confirmation dialog, click **Suspend** to restrict the vendor's access.

**F. Reactivate Vendor**

1. Click the **Check Circle** icon in the actions column for a suspended vendor.
2. In the confirmation dialog, click **Reactivate** to restore the vendor's access.

**G. Access Guide Tour**

1. On the **Vendors** page, click the **Question Mark (Help)** icon next to the page title.
2. Follow the interactive steps to learn about vendor management.

## 4. Category Management Workflow

This workflow describes how an admin manages food categories: creating new categories with icons, updating existing ones, and toggling their active status in the system.

**A. Create Category**

1. On the **Category** page, click the **Add Category** button.
2. In the modal, enter the **Category Name** and **Description**.
3. Click the image upload area to select a category icon.
4. Click **Save** to create the category.

**B. View Categories**

1. Navigate to the **Category** page from the sidebar.
2. View the list of categories with their icons, names, descriptions, and current status.

**C. Update Category**

1. Click the **Edit (Pencil)** icon in the actions column of a category.
2. In the modal, modify the name, description, or update the icon image.
3. Click **Save** to apply changes.

**D. Deactivate / Activate Category**

1. Click the **Block** icon to deactivate an active category or the **Check Circle** icon to activate a deactivated one.
2. Confirm the action in the dialog.

**E. Access Guide Tour**

1. On the **Category** page, click the **Question Mark (Help)** icon next to the page title.
2. Follow the interactive steps to learn about category management.

## 6. User Management Workflow

This workflow describes how an admin manages different types of users: searching for users, monitoring customer tiers, banning/unbanning accounts, and promoting customers to moderators.

**A. View User List by Role**

1. Navigate to **User Management** and select **Customer**, **Partner (Vendor)**, or **System** from the sidebar.
2. View the list of users belonging to the selected role category.
3. Observe the **Tier (Hạng)** column for Customers to see their rank (Bạc, Vàng, Kim cương, or Cảnh báo).

**B. Search Users**

1. Click on the search bar in the **Bộ lọc** section.
2. Enter a username or email to filter the user list.

**C. Ban / Unban User**

1. Click the **Chặn (Block)** button for an active user or **Bỏ chặn (Unlock)** for a banned user.
2. In the confirmation dialog, click the confirm button (Chặn or Bỏ chặn) to update the status.

**D. Promote to Moderator (Customer Only)**

1. For a **Customer** user, click the **Thêm Moderator** button in the actions column.
2. In the confirmation dialog, click **Thêm Moderator** to grant moderator privileges.

**E. Access Guide Tour**

1. On any **User Management** page, click the **Question Mark (Help)** icon next to the page title.
2. Follow the interactive steps to learn about user management features.

## 8. Badge Management Workflow

This workflow describes how an admin manages badges and assigns them to users: creating badges with icons, updating badge info, and awarding or revoking badges for specific users.

**A. Create Badge**

1. Navigate to **Badge** > **Manage Badge** from the sidebar.
2. Click the **Add Badge** button.
3. In the modal, enter the **Badge Name**, **Description**, and upload an **Icon**.
4. Click **Save**.

**B. Update / Status Badge**

1. On the **Manage Badge** page, use the **Edit** icon to modify badge details.
2. Use the **Block/Check** icons to change the badge's active status.

**C. Award / Revoke Badge from User**

1. Navigate to **Badge** > **User Badge** from the sidebar.
2. Click the **Thêm hoặc thu hồi huy hiệu** button for a specific user.
3. To award: Select a badge from the dropdown and click **Gán huy hiệu**.
4. To revoke: Click the **X** icon on an assigned badge in the modal list.

**D. Access Guide Tour**

1. On either badge management page, click the **Question Mark (Help)** icon.
2. Follow the interactive steps to learn about badge features.

## 10. Campaign Management Workflow

This workflow describes how an admin manages system-wide and vendor-initiated campaigns: creating complex campaigns with quests and vouchers, monitoring branches, and reviewing vendor campaigns.

**A. Create System Campaign (with Quest/Voucher Bundle)**

1. Navigate to **Campaign** > **System Campaign** from the sidebar.
2. Click **Thêm chiến dịch**.
3. Enter campaign details (Name, Dates, Description).
4. Scroll to the **Quest & Voucher Bundle** section to add initial missions and rewards for the campaign.
5. Click **Save**.

**B. View Campaign Details & Branches**

1. Click the **Visibility** icon in the actions menu of a campaign to see detailed configuration.
2. Click the **Storefront** icon to see the list of participating branches and their status.

**C. Manage Campaign Quests & Vouchers**

1. Click the **Assignment** icon to open the quest management modal for a specific campaign.
2. Click the **Voucher** icon to manage discount vouchers associated with the campaign.

**D. Update Campaign**

1. On the **System Campaign** page, click the **Edit (Pencil)** icon in the actions menu of a campaign (available only for updateable campaigns).
2. Modify the campaign's name, dates, or description in the form.
3. To update the campaign image, upload a new file in the image section.
4. Click **Save** to apply the changes.

**E. Manage Campaign Vouchers**

1. On the **System Campaign** page, click the **Voucher (Ticket)** icon in the actions menu.
2. View the list of vouchers assigned to the campaign, including their codes, types, and discount values.
3. Use the **Edit** icon to modify campaign-specific vouchers or the **Visibility** icon to view details.

**F. View Vendor Campaigns**

1. Navigate to **Campaign** > **Vendor Campaign** from the sidebar.
2. Use the **Autocomplete** search to select a **Vendor**.
3. Browse campaigns created by the selected vendor, using search and status filters.

**G. Access Guide Tour**

1. On either campaign page, click the **Question Mark (Help)** icon.
2. Follow the interactive steps to learn about campaign management.

## 12. Quest Management Workflow

This workflow describes how an admin manages system tasks and rewards: creating quests with specific tasks, monitoring participants, and managing quest availability.

**A. Create / Update Quest**

1. Navigate to the **Quest** page from the sidebar.
2. Click **Thêm nhiệm vụ** or the **Edit** icon.
3. Configure quest info and upload an image.
4. Add specific **Tasks** by choosing from types: **Order Count**, **Order Amount**, **Tier Up**, **Visit Count**, or **Feedback Count**.
5. Assign **Rewards** (Vouchers or Points) to each task.
6. Click **Save**.

**B. View Participants & Progress**

1. Click the **Groups** icon for a quest with participants.
2. View the list of users, including their Tier, Points, and XP.
3. Observe the detailed **Progress Bar** and individual **Task Completion** status for each participant.

**C. Status Management**

1. Use the **Block/Check** icons for quests with zero participants to toggle availability.

**D. Access Guide Tour**

1. On the **Quest** page, click the **Question Mark (Help)** icon.
2. Follow the interactive steps to learn about quest management.

## 13. System Setting Workflow

This workflow describes how an admin configures system-wide parameters: updating values inline and reloading configuration for the runtime environment.

**A. Update Setting Value**

1. Navigate to the **Setting** page from the sidebar.
2. Click the **Edit** icon next to a setting value.

4. Click the **Check** icon to save.

**B. Reload Runtime Settings**

2. The system will refresh its runtime configuration from the database.

**C. Access Guide Tour**

1. On the **Setting** page, click the **Question Mark (Help)** icon.
2. Follow the interactive steps to learn about system configuration.
